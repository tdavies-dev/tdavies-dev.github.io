# Tom Davies - Portfolio

A modern, responsive portfolio website built with React, TypeScript, Tailwind CSS, and shadcn/ui components.

## Features

- Single-page scrolling design
- Hero section with introduction
- Experience timeline with employment history
- Projects showcase section
- Fully responsive design
- Modern UI with shadcn components

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development Server

```bash
npm run dev
```

Visit `http://localhost:5173` to view the site.

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```

## Customization

- Edit `src/App.tsx` to update content, projects, and personal information
- Modify `src/index.css` to adjust the color scheme
- Add your profile picture by replacing the placeholder in the hero section
- Fill in your projects in the Projects section

## Tech Stack

- **React 18** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **shadcn/ui** - UI components
- **Lucide React** - Icons
